## API

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| icon | 设置头像的图标类型，参考 `Icon` 组件 | String | - | - |
| shape | 指定头像的形状 | String | circle / square | circle |
| size | 按钮的大小 | String | default / small / large | - |
| src | 图片类头像的资源地址 | String | - | - |
