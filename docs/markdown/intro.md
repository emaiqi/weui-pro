# 介绍

----

WeUI Pro 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，开发和服务于微信小程序。

## 特性

- 基于 `Vue` 开发的 UI 组件库，开箱即用
- 提供 `30+` 实用组件，能够满足小程序开发的基本需求
- 使用 npm + webpack + babel 的工作流，支持 ES2015
- 提炼自 `weui-wxss` 基础样式库，为微信小程序开发量身设计
- 提供友好的 API，可灵活的使用组件
- 完整的 `Vue.js` 开发体验
